## :rocket: Getting started

### 1. Clone or download this repository:

```sh
git clone https://github.com/josephgattuso/robofriends.git
```

### 2. Install dependencies:

```sh
yarn
```

### 3. Develop the app locally:

```sh
yarn start
```
